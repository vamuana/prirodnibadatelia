# Prírodní bádatelia

Vzdelávacia hra pre deti (4–8) – sekvencie, podmienky, cykly cez príbeh so zvieratkami v lese.

## Tech
- Unity 2022 LTS (URP), Git LFS
- Grid-based gameplay + blokový „programátor“

## Štruktúra (bude)
- `Assets/Art` – modely, materiály
- `Assets/Code` – Core, Interpreter, UI blokov
- `Assets/Content/Levels` – JSON levely
- `Assets/Resources/Localization` – jazyky
